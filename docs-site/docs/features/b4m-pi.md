# B4M Pi - Project Intelligence

*(Experimental Feature — enable in [Profile > Settings > Experimental Features](./profile-settings.md#experimental-features))*

B4M Pi is a project intelligence dashboard that provides repository analysis, task scheduling, Gantt chart visualization, and team activity tracking — all powered by your connected GitHub repositories.

> **Availability:** This feature may be enabled or disabled at the organizational level by your administrator. If the toggle is grayed out with "Disabled by administrator," contact your organization admin to request access.

---

## How to Enable

1. Click your **avatar** in the sidebar footer to open your Profile
2. Go to the **Settings** tab
3. Scroll to **Experimental Features**
4. Toggle **B4M Pi** on

## How to Access

Once enabled, B4M Pi is available from the sidebar footer:

1. In the left sidebar footer, click the **three-dot menu** (more options button)
2. Select **B4M Pi** from the menu
3. The full B4M Pi dashboard opens with all tabs available

## Dashboard Tabs

B4M Pi provides a full-featured project intelligence dashboard with the following views:

### Repos

Analyze GitHub repository activity — commit frequency, contributor metrics, and codebase health indicators. Requires a connected GitHub integration.

### My Activity

Track your personal development activity and contributions across repositories. See your commit history, pull request involvement, and coding patterns.

### Team Activity

View team-wide activity dashboards showing how the team is contributing across repositories. Compare activity levels and identify collaboration patterns.

### Tasks & Estimates

Manage tasks and create effort estimates. Organize work items and track progress across your projects.

### Solver & Gantt

Visualize project timelines with interactive Gantt charts. The scheduling solver helps optimize task sequencing based on dependencies and resource constraints.

### Resources

View resource allocation and team workload distribution. Understand how team members are distributed across active work streams.

### Insights

Analytics and derived insights from your project data — trends, bottlenecks, and patterns that help you understand project health.

---

## Prerequisites

- A connected **GitHub** integration (Settings > Connected Apps) for repository data
- Organization membership for team-level views
