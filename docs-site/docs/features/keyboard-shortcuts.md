# Keyboard Shortcuts

Quick reference for all keyboard shortcuts available in Bike4Mind.

---

## Global Shortcuts

These shortcuts work from anywhere in the app, as long as you are not focused inside a text input.

| Shortcut | Action |
| -------- | ------ |
| `?` | Open / close the Help Center |
| `Cmd` + `K` / `Ctrl` + `K` | Open the Command Palette |

---

## Chat Input

Shortcuts available while composing a message in the chat input.

| Shortcut | Action |
| -------- | ------ |
| `Enter` | Send message |
| `Shift` + `Enter` | Insert a new line |

---

## Tavern Canvas

Shortcuts available when working in the Tavern canvas view.

| Shortcut | Action |
| -------- | ------ |
| `W` / `A` / `S` / `D` | Pan the camera up / left / down / right |
| `↑` / `←` / `↓` / `→` | Pan the camera (arrow keys) |
| `E` | Toggle the tile editor |
| `H` | Hilltop view — zoom out to fit the entire map |
| `F` | Focus on the tavern — zoom in to room view |
| `Escape` | Clear selection; zoom out to street level if zoomed in |
| `Cmd` + `Z` / `Ctrl` + `Z` | Undo |
| `Cmd` + `Shift` + `Z` / `Ctrl` + `Shift` + `Z` | Redo |

---

## General UI

| Shortcut | Action |
| -------- | ------ |
| `Escape` | Close the current modal or panel |
