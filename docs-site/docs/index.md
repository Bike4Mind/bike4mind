# Welcome to Bike4Mind Documentation 🚀

**Your bicycle for the mind in the age of AI.** Bike4Mind is an open-core AI knowledge platform - a multi-model workspace where notebooks, autonomous agents, and a RAG knowledge engine work together to augment any cognitive task.

## 🎯 What is Bike4Mind?

- **🤖 Multi-LLM by design** - swap freely between Anthropic, OpenAI, Google Gemini, xAI, and local models via Ollama, plus image models
- **🧠 Intelligent Agents** - autonomous ReAct-style agents for complex workflows
- **⚔️ Quest Master** - autonomous multi-step task planning with human-in-the-loop steps
- **📚 Knowledge Engine** - RAG over your documents: smart chunking, vector search, collections, and tagging
- **🎨 Artifacts** - reusable snippets, documents, and visualizations produced by agents
- **🔧 Developer Tools** - REST APIs, a CLI, and integration frameworks

Try the hosted service at [app.bike4mind.com](https://app.bike4mind.com), or run the open core yourself - see [Self-Hosting](/self-host).
