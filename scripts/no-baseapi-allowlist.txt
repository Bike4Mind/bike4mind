# no-baseapi-allowlist.txt
#
# API files that legitimately skip baseApi() because they use an alternative
# authentication mechanism. Every entry must have a comment explaining why.
#
# To add a new entry:
#   1. Confirm the file uses a real alternative auth (HMAC, OAuth state, encrypted token, or intentionally public)
#   2. Add the path below with a comment explaining the mechanism
#   3. Get the entry reviewed in the PR
